# Pipeline Profiles

Select profile based on time and data availability.

## quick

- Scope: latest quarter + latest annual filing
- Goal: fast triage for obvious red flags
- Output: abbreviated risk register and watchlist

## standard

- Scope: latest 8 quarters + latest annual filing
- Goal: balanced signal quality and turnaround
- Output: full 5-file output bundle

## deep

- Scope: multi-year review including major M&A windows and restatement periods
- Goal: high-confidence pattern detection and regime-shift analysis
- Output: full bundle plus expanded open questions

## Stop Conditions

Stop and report blockers when:

- Core statements or key footnotes are missing
- Period alignment cannot be reconciled
- KPI definitions change without sufficient disclosure to restate history

## Confidence Downgrade Triggers

Downgrade any risk call one level when:

- Numeric signal is not reproducible from filing numbers
- Counter-evidence from same filing period remains unresolved
- Management disclosure directly explains the anomaly and the data supports it
